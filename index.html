Free Trial & Recurring Billing Flows

1
7-Day Free Trial

New subscriptions include automatic 7-day free trial → Stripe sends trial_will_end webhook → viax sends reminder email → trial ends and first payment processed

↓
2
Recurring Billing Success

Stripe bills customer → payment.succeeded webhook to viax → viax maintains license via License Manager API → confirmation email via Segment

↓
3
Recurring Billing Failure

Stripe retry attempts → payment.failed webhook to viax → grace period begins → reminder emails → if still failed, viax revokes license → final notification

Open Question: Exact duration of grace period? How many retry attempts before license revocation? Do users retain course access during retry window or immediately revoked?
